Configure o NativeWind v4 neste projeto Expo. Siga exatamente estes passos:

1. Instale as dependências:
   npm install nativewind tailwindcss

2. Crie o arquivo `tailwind.config.js` na raiz com:
   - content apontando para "./app/**/*.{js,jsx,ts,tsx}" e "./src/**/*.{js,jsx,ts,tsx}"
   - presets com require("nativewind/preset")

3. Crie o arquivo `global.css` na raiz com os imports do Tailwind (@tailwind base, components, utilities)

4. Edite o `babel.config.js` para incluir:
   - preset "babel-preset-expo" com { jsxImportSource: "nativewind" }
   - preset "nativewind/babel"

5. Crie o arquivo `metro.config.js` usando getDefaultConfig do "expo/metro-config" e withNativeWind do "nativewind/metro", passando { input: "./global.css" }

6. Crie o arquivo `nativewind-env.d.ts` com a referência de tipos do NativeWind

7. No arquivo de layout principal (app/_layout.tsx), adicione o import do "../global.css" no topo

8. Após finalizar, crie um componente de teste no app/index.tsx com algumas classes do Tailwind (ex: bg-blue-500, text-white, p-4) para verificar se está funcionando

Não pule nenhum passo e confirme cada arquivo criado/editado.
